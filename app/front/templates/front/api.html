Introducción

Bienvenido a la documentación de la api v1. Puedes utilizar esta API para obtener información sobre los académicos y sobre la institución a la que pertenecen.

Tenga en cuenta que, aunque nos esforzamos por mantener actualizada esta documentación, considérela un trabajo en curso y tenga en cuenta que es probable que contenga errores.

Los ejemplos de código en bash se proporcionan en la zona oscura de la derecha.

Los ejemplos de código en bash se proporcionan en la zona oscura de abajo.

Condiciones de uso

Se espera que la API sea utilizada para hacer el bien. No se exceda. En caso de duda, pregunte antes de sobrecargar el sistema. Esta sección puede ampliarse según sea necesario.

Endpoint

URL base

La URL base es: https://{{ request.get_host }}/api/v1/

Académicos

Objeto Académico

Objeto de respuesta entregada para representar un académico. Contiene los siguintes atributos:

Atributo Descripción
id (int) ID de académico en el sistema.
nombre (string) Nombre de académico registrado.
apellido (string) Apellidos de académico registrado
email (string) Correo electrónico del académico registrado.
webpage (string) Página web del académico registrado.
Grado Máximo (string) Grado Máximo alcanzado por el académico (TECH,LIC,LSC,PHD)
afiliacion (string) Objeto con la información de la afiliación del académico
Atributo Descripción
unidad (string) Data relacionada a la unidad a la que pertenece el académico (Nombre, Sigla, Web).
universidad (string) Data relacionada a la Institución a la que pertenece el académico (Nombre, Pais, Acronimos, Web).
IDs Externos (string) Objeto con los identificadores externos obtenidos para el académico
Atributo Descripción
dblp_id (string) Identificador ORCID del académico.
aminer_id (string) Identificador DBLP del académico.
openalex_id (string) Identificador DBLP del académico.
orcid_id (string) Identificador DBLP del académico.
areas_trabajo (string) Lista de objetos con las áreas, subáreas y keywords en las que el académico realiza investigación
Atributo Descripción
area (string) Objeto con la información del área
  • ID: ID del área.
  • Nombre: Nombre del área.
  • Subáreas: Lista de objetos con las subáreas relacionadas al área.
Keywords (string) Objeto con la información de la subárea (ID, Nombre, Keywords).

Búsqueda básica por nombre

Este endpoint permite buscar académicos basándose en búsquedas simples por nombre. Estos pueden tener un margen de error, por lo que el resultado a entregar es un listado de posibles candidatos ordenados por el nombre que contenga mayor similitud.

Request

GET https://{{ request.get_host }}{% url 'api:academico_search_basic' %}

Headers

Content-Type: application/json

Accept: application/json

Query Parameters

query string

String con el nombre a buscar.

Response format

Retorna un listado de objetos académicos.

‌

Ejemplo request:

curl --request GET \

    --get "https://{{ request.get_host }}{% url 'api:academico_search_basic' %}?query=Ignacio%20Cuellar" \

    --header "Content-Type: application/json" \

    --header "Accept: application/json"

Ejemplo response (200):

{

    "academicos": [

        {

            "id": 1,

            // Otros atributos...

        },

        ...

    ]

}

Búsqueda avanzada

Este endpoint permite buscar académicos basándose en búsquedas por nombre y filtros. La búsqueda por nombre puede tener un margen de error, por lo que el resultado a entregar es un listado de posibles candidatos ordenados por el nombre que contenga mayor similitud. Los filtros disponibles son por institución, país, áreas y subáreas.

Request

GET https://{{ request.get_host }}{% url 'api:academico_search_advanced' %}

Headers

Content-Type: application/json

Accept: application/json

Query Parameters

nombre string opcional

String con el nombre a buscar.

institucion string opcional

String con el nombre de institución a la que pertenece el académico.

pais ISO 3166-1 opcional

Código ISO 3166-1 del país al que pertenece la institución del académico.

area string opcional

Área en la cual el académico desempeña sus investigaciones.

subarea string opcional

Subárea en la cual el académico desempeña sus investigaciones.

Response format

Retorna un listado de Objetos Académico

‌

Ejemplo request:

curl --request GET \

    --get "https://{{ request.get_host }}{% url 'api:academico_search_advanced' %}?nombre=Ignacio%20Cuellar&pais=CL" \

    --header "Content-Type: application/json" \

    --header "Accept: application/json"

Ejemplo response (200):

{

    "academicos": [

        {

            "id": 1,

            // Otros atributos...

        },

        ...

    ]

}

Obtención por ID

Este endpoint permite extraer toda la información que se tiene sobre un académico en específico. Esta petición es realizada por cualquier ID que este académico posea, ya sea el identificador otorgado por el sistema mismo, el identificador de DBLP o el identificador de ORCID.

Request

GET https://{{ request.get_host }}{% url 'api:academico_get_id' %}

Headers

Content-Type: application/json

Accept: application/json

Query Parameters

id int opcional

Identificador otorgado por el sistema.

dblp string opcional

Identificador de DBLP

orcid string opcional

Identificador de ORCID.

Response format

Retorna un único Objeto Académico.

‌

Ejemplo request:

curl --request GET \

    --get "https://{{ request.get_host }}{% url 'api:academico_get_id' %}?id=12452" \

    --header "Content-Type: application/json" \

    --header "Accept: application/json"

Ejemplo response (200):

{

    "id": 1,

    // Otros atributos...

}

Institución

Objeto Institución

Objeto de respuesta entregada para representar una institución. Contiene los siguintes atributos:

Atributo Descripción
id (int) ID de institución en el sistema.
nombre (string) Nombre de institución registrado.
siglas (list) Listado de siglas referidad a la institución.
pais (string) País donde se encuentra la institución.
webpage (string) Página web de la institución.
ids_externos (string) Objeto con los identificadores externos obtenidos para la institución
Atributo Descripción
grid_id (string) Identificador GRID de la institución.
ror_id (string) Identificador ROR de la institución.
mag_id (string) Identificador MAG de la institución.
openalex_id (string) Identificador OpenAlex de la institución.
geo_data (string) Objeto con los datos geográficos de la institución
Atributo Descripción
latitud (float) Latitud de la institución.
longitud (float) Longitud de la institución.
ciudad (string) Ciudad donde se encuentra la institución.

Programas

Objeto Programa

Objeto de respuesta entregada para representar un programa académico. Contiene los siguintes atributos:

Atributo Descripción
id (int) ID del programa en el sistema.
nombre (string) Nombre del programa registrado.
tipo (string) Tipo de programa (TECH, LIC, MSC, PHD).
afiliacion (object) Objeto con la unidad y universidad a la que pertenece el programa.
Atributo Descripción
unidad (object) Objeto Unidad (ver sección Unidad para más detalles).
universidad (object) Objeto Universidad (ver sección Institución para más detalles).
web_site (string) Página web del programa.
fecha_creacion (string) Fecha de creación del programa (YYYY-MM-DD).

Búsqueda básica

Búsqueda simple de programas por nombre. Usa búsqueda difusa y devuelve resultados ordenados por similitud.

Request

GET https://{{ request.get_host }}{% url 'api:grado_search' %}

Query Parameters

query string

Texto para búsqueda difusa del nombre del grado.

Response format

Devuelve una lista de Objetos Programa .

Ejemplo request:

curl --request GET \

    --get "https://{{ request.get_host }}{% url 'api:grado_search' %}?query=master" \

    --header "Accept: application/json"

Ejemplo response (200):

{

    "results": [

        {

            "id": 10,

            "nombre": "Master in Computer Science",

            "afiliacion": {"unidad": {...}, "universidad": {...}}

        },

        ...

    ]

}
